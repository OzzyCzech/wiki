Desktop and mobile clients for browsing and querying databases.

## Native

- **[TablePro](https://tablepro.app/)** — native macOS 14+ / iOS 17+ database client supporting 18+ engines (MySQL, PostgreSQL, MongoDB, Redis, SQLite, …); SQL editor with autocomplete and Vim mode, ER diagrams with crow's foot notation, SSH tunneling, server dashboard with active session and slow query monitoring, AI assistant for natural-language queries, built-in terminal for native CLIs (`mysql`, `psql`, `redis-cli`), iCloud sync between Mac and iPhone, Safe Mode with Touch ID for production databases, CSV/JSON/SQL/XLSX import-export
